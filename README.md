# Plum
De-Dockerized coding style checker for Epitech projects


## Install

```sh
curl https://raw.githubusercontent.com/LouisDupraz/Plum/main/plum_install.sh | bash
```
### Requires
- Python3
- Boost libs
- Lua
- Docker (Only for installation)

## Usage

```sh
plum [OPTIONS]

Options:
  --update  Update plum
```
